This project was developed as part of a college unit focused on web development. The objective was to design and build a prototype web page using HTML, CSS, and Bootstrap to demonstrate understanding of layout, styling, and responsive design principles.

The repository contains several HTML files including pages for college, gaming, social, and sports, along with a dedicated CSS stylesheet and an images folder. Each page was created to explore different aspects of front‑end development and to apply consistent design through reusable styles and responsive layouts.

This prototype represents my early experience in web development and showcases my ability to structure and style content effectively while maintaining clarity and accessibility.
